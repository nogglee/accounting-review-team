회계 자료를 분석하여 경영자가 빠르게 판단할 수 있는 재무 진단 보고서를 자동으로 생성하는 Claude CoWork 기반 에이전트 팀입니다.

---

# 이 프로젝트가 필요한 사람

다음 중 하나라도 해당된다면, 이 프로젝트가 정확히 찾던 도구입니다:

- 매월 회계 자료는 받지만, 무엇을 봐야 하는지 모르겠다
- 회계사가 준 자료를 "읽는" 대신 "판단" 가능한 보고서가 필요하다
- 비용 이상 증가, 누락, 세무 리스크를 자동으로 탐지하고 싶다
- 경영자 관점에서 요약된 재무 상태 보고서를 자동으로 생성하고 싶다
- 회계 데이터를 기반으로 빠르게 의사결정을 내려야 한다

이 프로젝트는 다음을 자동으로 수행합니다:

- 회계 자료 분석
- 이상 징후 탐지
- 세무 리스크 식별
- 누락 및 분류 오류 검토
- 경영자용 PDF 보고서 생성

결과적으로, 경영자는 원본 회계 자료를 직접 해석할 필요 없이
"지금 무엇을 확인하고 결정해야 하는지"만 보면 됩니다.

---

# 이 프로젝트가 필요하지 않은 경우

다음 목적이라면 이 프로젝트는 적합하지 않을 수 있습니다:

- 단순 회계 장부 기록 시스템이 필요한 경우
- 회계 처리(기장)를 자동화하려는 경우
- ERP 또는 회계 프로그램을 대체하려는 경우

이 프로젝트는 회계 시스템이 아니라,
"회계 자료를 해석하여 경영자용 보고서를 생성하는 분석 시스템"입니다.

---

# 사용법

1. 초록색 버튼을 클릭하고 Download zip을 클릭하세요.
2. 다운로드 받은 파일의 압축을 해제해주세요.
3. 클로드 코워크 실행하세요.
4. 작업할 폴더로 압축을 해제한 폴더를 선택해 주세요.
5. 분석이 필요한 자료를(이미지, PDF) 클로드 코드에 업로드하고, "분석해줘"를 요청하세요.
6. 7~10분 내로 결과를 받아볼 수 있습니다.

---

# 이 프로젝트가 제공하는 결과

입력:

- 합계잔액시산표
- 손익계산서
- 재무상태표

출력:

```
outputs/2025-09/
  2025-09-경영자보고서.pdf
```

보고서에는 다음이 포함됩니다:

- Executive Summary (30초 요약)
- 핵심 리스크
- 이상 징후
- 권장 조치(Action Plan)

경영자는 이 보고서 하나만 보면 현재 재무 상태를 빠르게 판단할 수 있습니다.

---

이 시스템은 합계잔액시산표, 손익계산서, 재무상태표 등의 회계 자료를 분석하고, 구조화된 검증(JSON)과 최종 경영자 보고서(PDF)를 생성합니다.

---

# 목적

경영자는 회계 자료를 "보관"하는 것이 아니라 "판단"하기 위해 사용합니다.

하지만 일반적인 회계 자료는:

- 전문 용어 중심
- 문제 우선순위 불명확
- 의사결정 관점 부족

이 프로젝트의 목적은 다음입니다:

- 회계 자료 → 경영자용 의사결정 보고서로 변환
- 리스크, 이상 징후, 누락 가능성 자동 탐지
- 빠른 판단을 위한 정형화된 보고서 생성
